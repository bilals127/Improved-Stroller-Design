# Improved-Stroller-Design

- What? Design a human-powered transport vehicle accommodating user and co-user bases with musculoskeletal disorders, post-surgery needs, vibration sensitivity
- How? Completed as part of a semester-long team project with 5 other students
- Conducted systems and human factors analysis to quantify energy and mass flows, identify required design changes
- Created associated technical drawings, parts, and assemblies in SOLIDWORKS
- Personal focus on an augmented caster with a suspension system
